# Vite Lightning CSS Plugin

## Installation

### Vite

```bash
npm i -D vite-plugin-lightningcss
```

```ts
// vite.config.ts
import lightningcss from "vite-plugin-lightningcss";

export default {
  plugins: [
    lightningcss({
      /* options */
    }),
  ],
};
```

That's it!
